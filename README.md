[React Query](https://react-query.tanstack.com/overview)

### Start Project:
⭐ `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Test Project:
⭐ `yarn test`
### Build Project: 
⭐ `yarn run build`
